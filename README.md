# AiTaskDashboard

An advanced, AI-powered task management dashboard designed to streamline productivity and innovation workflows.

## Features

*   **Intelligent Task Management:** Visualize and organize tasks with ease using our intuitive dashboard.
*   **Innovation Pipeline:** Track ideas from conception to implementation with a dedicated pipeline view.
*   **Real-time Analytics:** Gain insights into your project's progress with dynamic charts and metrics.
*   **Modern UI/UX:** Built with a focus on aesthetics and user experience, featuring a dark mode and responsive design.

## Tech Stack

*   **Frontend:** React, TypeScript, Vite
*   **Styling:** Vanilla CSS (with modern features like variables and flexbox/grid)
*   **Icons:** Lucide React
*   **Routing:** React Router DOM

## Getting Started

### Prerequisites

*   Node.js (v18 or higher)
*   npm or yarn

### Installation

1.  Clone the repository:
    ```bash
    git clone https://github.com/burnerdev42/AiTaskDashboard.git
    ```
2.  Navigate to the dashboard directory:
    ```bash
    cd dashboard
    ```
3.  Install dependencies:
    ```bash
    npm install
    ```

### Running the Application

To start the development server:

```bash
npm run dev
```

Open your browser and navigate to `http://localhost:5173` (or the port shown in your terminal).
