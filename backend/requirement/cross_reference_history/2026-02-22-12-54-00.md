# Cross-Reference: Centralized Hardcoded Constants

## Summary
Created `backend/src/common/constants/app-constants.ts` — centralized all 13 hardcoded constant groups (including `COMMENT_TYPES`).
All enum constraints in backend code now reference this single constants file instead of inline arrays.
